### Application that sends messages by ```GMAIL```.
---
### preview:
![screenshot](./screenshot_2021-07-16-220011.png)
#### Run app:
1. **export the following environment variables:**
    - ```export GMAIL='email@example.com'```
    - ```export PASS_GMAIL='secretpass'```
2. **execute command:** 
    - ```go run main.go```
